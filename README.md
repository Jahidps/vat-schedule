# VAT Schedule Generator

Interactive tool to produce PaymentSave VAT Schedule PDFs (standard monthly + interim).

## Use it live
Open: https://jahidps.github.io/vat-schedule/

Fill in the form, then click "Save as PDF / Print".

- Standard: enter the Tax Point / Commencement Date and Net; monthly rows run to December, VAT is always 20%, Gross calculated.
- Interim: enter the Gross; Net (Gross/1.20) and VAT (Gross-Net) are calculated. Tax Point = Period From.
